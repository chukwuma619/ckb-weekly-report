# Weekly report — 17 May 2026 to 23 May 2026

**Project:** `fiber-desktop` — Fiber Network desktop client (Tauri, React, Rust backend for FNN integration).

Github: https://github.com/chukwuma619/fiber-desktop

## Summary

**Payments UX** split into dedicated Receive and Send tabs, **FNN runtime hardening** (orphan process cleanup, audit fixes from review), a **major App refactor** into tab components and hooks, **cross-platform CI** for Windows/Linux/macOS, and a **documentation overhaul** (architecture, development, release QA, contributing).

**Also this week:** connect-peer setup panel with config sync, marketing guide updates for the new tab layout, download-page retry UX, and consolidated READMEs across the app and marketing site.

## What I shipped (high level)

1. **Payments UX** — Split the monolithic Payments tab into separate **Receive** and **Send** tabs; extracted shared RPC helpers (`cf5b067`, 2026-05-18).

2. **FNN runtime reliability** — Orphan FNN process detection and cleanup on app exit; merged runtime audit fixes from review (packaged sidecar lookup, settings/process lifecycle) via [PR #2](https://github.com/chukwuma619/fiber-desktop/pull/2) (`2cd1181`, `c71624f`, `edbc42a`, `225aa0c`, 2026-05-18–19).

3. **Setup & connect peer** — `SetupConnectPeerPanel` with inline feedback styles; backend `config_sync` to keep FNN config aligned with app settings (`89f7f9f`, 2026-05-18).

4. **App architecture refactor** — Broke the 1,300-line `App.tsx` into tab components (`OverviewTab`, `SetupTab`, `NodeTab`), hooks (`useAppSettings`, `useFnnBinary`, `useGuidedSetup`, `useCkbKey`, `useNodeRuntime`), and shared utilities; improved FNN binary management (`fca8b24`, 2026-05-22).

5. **Cross-platform support** — Windows path pickers, clipboard helpers, platform labels, and smoke-test docs; Linux CI job and platform-specific settings handling; macOS CI enhancements and Rust unit tests in CI (`37f3732`, `6141e0c`, `cb801cb`, 2026-05-22).

6. **Documentation** — Added `docs/ARCHITECTURE.md`, `docs/DEVELOPMENT.md`, `docs/CONTRIBUTING.md`, `docs/RELEASE_QA.md`, plus `LICENSE` and `SECURITY.md`; consolidated scattered READMEs and removed outdated smoke-test docs (`e8e4334`, `17e6797`, `fe0f345`, 2026-05-22–23).

7. **Marketing** — Updated guide pages for Receive/Send tab terminology; download page error handling with retry (`99cc634`, `3e7504c`, 2026-05-18).
